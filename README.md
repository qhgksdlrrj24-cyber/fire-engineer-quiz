# 소방기술사 학습 시스템 🔥

온라인에서 사용 가능한 소방기술사 주관식 학습 애플리케이션입니다.

## 기능

- 📚 **강의별 학습**: 0강부터 6강까지 단계별 학습
- ⭐ **별표 문제**: 중요한 문제 표시 및 복습
- ✍️ **주관식 답안 작성**: 직접 답안을 작성하고 정답 확인
- 📊 **학습 진도 관리**: 자동으로 학습 진행률 저장
- 🔄 **진도 초기화**: 학습 기록 리셋 기능

## 문제 수

- 0강: 94문제
- 1강: 109문제
- 2강: 94문제
- 3강: 95문제
- 4강: 115문제
- 5강: 66문제
- 6강: 131문제
- **총 704문제**

## 사용 방법

1. 학습 모드 선택 (강의별/별표/전체)
2. 원하는 강의 선택
3. 문제를 읽고 답안 작성
4. 정답 확인
5. 다음 문제로 이동

## 기술 스택

- HTML5
- CSS3 (Clean White Theme)
- Vanilla JavaScript
- LocalStorage (진도 저장)
